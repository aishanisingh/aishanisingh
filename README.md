## selected projects

- **liveconv** (2023) — real-time speech-to-speech translation running fully on ar glasses, using cascaded asr and attention-based translation models under tight latency budgets.
- **netra** (2022) — on-device ai navigation aid for visually impaired users; combines inceptionv3-with-attention image captioning with a wavenet text-to-speech pipeline on resource-constrained hardware. *synopsys championship 1st place · csef finalist.*
- **cpdm** (2023) — a 21,000-image benchmark for diffusion-model unlearning across four copyright categories.
